Une nouvelle version de Pentaho Data Integration est disponible:

Version $version$

Vous pouvez télécharger cette version depuis:
$url$

Remarque: Lors de la vérification d'une nouvelle version, Pentaho Data Integration se connecte au serveur Pentaho.
Aucune donnée sensible n'est transmise lors de cette vérification.
De plus amples informations sur le processus de vérification peuvent être trouvées sur la page wiki.
La vérification d'une nouvelle version est une option gérée depuis le menu Edition / Options
