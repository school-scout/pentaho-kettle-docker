Export to PDF and Save or Print

Complete the following steps to print a report.

  1. Open the report you want to print.
  2. Select Export » Export to PDF. (This will convert your report into a PDF file (Adobe Acrobat) and launch a new window.)
  3. Use the Adobe Print command (File » Print) or Save As command (File » Save As)in this window.

Export to Excel with Formatting

Complete the following steps to export a Pentaho report to Excel:

  1. Open the report you want to print.
  2. Select Export » Export to Excel. (This will convert your report into a .xls file (Microsoft Excel) and launch a new window.)

    3. Note: Excel 2008 limits reports to 256 columns and 10,000 rows.

Export Report Data to CSV

Complete the following steps to download report data into CSV format.

  1. Open the report whose data you want to download.
  2. Click the More button
  3. Select Export to CSV from the menu. (This will open an Excel sheet displaying your report data.)

Note: When you download data in the CSV format you will get numbers with the full precision available. This way you avoid any rounding errors when you continue to work with your data in Excel.

By default, your data will be saved in CSV (Comma Separated Values) format. If you'd like you can complete the following steps to convert the data from CSV format into a regular Excel file (XLS).

  1. Select File » Save As in Excel to open the Save As dialog box.
  2. Select Microsoft Excel (*.xls) from the Save as type menu.
  3. Click the Save button to save the file in XLS format.

Note: Only the data, with no report formatting, is transferred when you Export to CSV.
