About Hotkeys

Hotkeys are keyboard shortcuts. We have associated the most common report building actions with hotkeys to help you work more efficiently.

Pentaho Hotkeys

You can use hotkeys to conduct many of the most common actions. In most cases, the hotkey is indicated on the menu item or as a tool tip when hovering over the button.

The table below outlines all hotkeys.
Page / Element Action Hotkey
Report View Page Show / Hide Field List. Ctrl+Alt+F
Report View Page Show / Hide Filters. Ctrl+Alt+T
Report View Page Show / Hide Field Layout. Ctrl+Alt+Y
Report View Page Toggle between table format and default chart format. Ctrl+Alt+C
Report View Page Reset. Ctrl + Q
Report View Page Undo. Ctrl + Z
Report View Page Redo. Ctrl + Y
