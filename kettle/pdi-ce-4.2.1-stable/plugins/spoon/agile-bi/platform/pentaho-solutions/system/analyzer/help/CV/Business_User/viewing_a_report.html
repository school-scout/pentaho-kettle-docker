Finding a Report

To find a report, click the Home tab in the header to go to the Home page. Reports are grouped into:

  • My Reports: These reports are only visible to you (and administrators).
  • Shared Reports: These reports can be visible to multiple users.

Your administrator controls what you can view in the solution repository and which folders you can access. For more information, please see Security and Access Rights.

Viewing a Report

To view one of the reports listed on the Home page, simply click the name of the report. For more information on how to modify the report and analyze the data, see About Reports.

Viewing Information About a Report

To view additional information about a report, do the following:

  1. Open the report for which you want to view additional information.
  2. Click the Information symbol next to the name of the report to open the About This Report dialog box.

The About This Report dialog box provides the following information:

  • Name
  • Folder
  • Analysis Area
  • Based on Data as of (This field tells you when data was last loaded into Pentaho for each data source that might contribute data to the report. For example, data could be loaded from Salesforce.com or NetSuite.)
  • Date Modified
  • Modified By
  • Date Created
  • Description

Note:You may edit the Description field, but modifications of the description are not saved until you save the report using the Save or Save As command.

Saving a Report

To save a report using the existing name, click the Save button.

To save your report with a different name, select Save As... from the Save menu.

Depending on how your administrator has set you up, you might only be able to save reports in certain folders.

Note: When you save your report, it is stored in a folder in Pentaho's system, so you can access your report from any computer. When you save your report, the actual data in the report is not saved. Rather, the next time you open the report, it will retrieve the latest data from the database. However, you can store a static ("snapshot") version by storing the report as a PDF or Excel file on your computer. See Exporting Reports.

Deleting a Report

To delete a report, click the Delete symbol (X) next to the report in the list on the Report Home page.

Note: Depending on the permissions your administrator has given you, you might not be able to delete certain reports. In those cases, the Delete symbol will not be available to you.
